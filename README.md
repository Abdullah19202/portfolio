# Abdullah Wajdan — Portfolio

Simple static portfolio built with HTML, CSS, and JavaScript.

## Local preview
Open `index.html` in a browser or run a local server, for example with Python 3:

```bash
python -m http.server 5500
# then visit http://localhost:5500
```

## Deploy to GitHub Pages
This repository is ready for GitHub Pages. Push to the `main` branch and enable Pages in repository settings, or use the included GitHub Actions workflow which deploys on push to `main`.

Notes:
- If you want a custom domain, add a `CNAME` file to the repository root and update DNS.
- The workflow uses GitHub's Pages actions and requires no extra secrets.
